# Test Calculateur Geovelo

## A noter

Il est nécessaire de disposer d'une clé API PRIM. Cette clé est à insérer dans le script `test.py` (variable `API_KEY`).

Il s'agit d'un test exploratoire de l'API du calculateur Géovélo pour les besoins du projet Rayon d'action.

## Usage

```bash
usage: Script description [-h] [--lon_start LON_START] [--lat_start LAT_START] [--lon_end LON_END] [--lat_end LAT_END] [--instructions] [--elevations] [--geometry]
                          [--single_result] [--bike_stations] [--objects_as_ids] [--merge_instructions] [--show_pushing_bike_instructions]

options:
  -h, --help            show this help message and exit
  --lon_start LON_START
                        lon start
  --lat_start LAT_START
                        lat start
  --lon_end LON_END     lon end
  --lat_end LAT_END     lat end
  --instructions        instructions
  --elevations          elevations
  --geometry            geometry
  --single_result       single_result
  --bike_stations       bike_stations
  --objects_as_ids      objects_as_ids
  --merge_instructions  merge_instructions
  --show_pushing_bike_instructions
                        show_pushing_bike_instructions
```
